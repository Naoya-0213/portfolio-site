制作課題

アクセサリーブランド「sabolon」架空サイト

概要

スクール（デイトラ）の課題として、架空のアクセサリーブランド「sabolon」のwebサイトを制作しました。デザインカンプをもとに、視覚的な魅力の強化とユーザーエクスペリエンスの向上を意識し、細部にこだわりながら丁寧に実装しています。

ターゲット

製品の購入を検討している既存顧客、およびアクセサリーの購入を検討している新規顧客。（想定）

課題

「sabolon」には製品専用のサイトや製品への想いを伝える場がなく、ブランドの魅力が十分に伝わっていなかった。そのため、購入機会の損失が目立っていた。（想定）

目的

製品専用サイトを通じて認知度を高めると同時に、ブランド理念に共感してもらうことで、既存顧客の維持はもちろん、新規顧客の獲得を目指す。（想定）

担当領域

デザインカンプからのコーディング
サイト更新＆修正

制作期間

コーディング　　　1週間
サイト更新＆修正　1週間

使用ツール

Figma / Visual Studio Code
